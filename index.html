j'ai besoin que mes listes soient repliées dans ce fichier
ChatGPT a dit :

Pour que les listes de votre fichier HTML soient repliées par défaut, il faut que les éléments <details> n’aient pas l’attribut open. Actuellement, vos blocs <details> sont ouverts par défaut à cause de la présence de open="".

✅ Solution

Remplacez :

html
<details open="">

par :

html
<details>

🔧 Exemple avant/après

Avant :

html
<details open="">
  <summary>Ma question</summary>
  <p>La réponse</p>
</details>

Après (replié par défaut) :

html
<details>
  <summary>Ma question</summary>
  <p>La réponse</p>
</details>

📁 Application à tout le fichier

Vous pouvez faire une recherche/remplacement global dans un éditeur de texte :

  • Rechercher : <details open="">

  • Remplacer par : <details>

Cela repliera toutes les sections de type accordéon (<details>) dans votre FAQ.
